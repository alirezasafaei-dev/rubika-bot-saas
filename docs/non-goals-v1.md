# A05 Non-Goals v1

This document freezes what `A05` does not try to solve in the current delivery model.

## Product Non-Goals

- No public multi-tenant SaaS launch
- No frontend admin panel
- No subscription billing
- No AI assistant or generative features
- No voice, OCR, or media-processing automation
- No advanced analytics beyond the current reports scope
- No CRM, campaign manager, or sales pipeline module
- No third-party authentication providers
- No mandatory cloud runtime dependencies
- No cross-platform bot framework beyond Rubika MVP scope

## Delivery Non-Goals

- No unmanaged custom feature growth during deployment
- No undocumented production changes
- No direct use of real secrets inside the repository
- No support promise without a defined support agreement

## Decision Rule

If a request conflicts with this list, it is outside v1 and must be handled as a separate scoped engagement.
