# A05 Pricing and Scope Matrix

## Packages

| Package | Best For | Scope | Delivery Window | Starting Price |
| --- | --- | --- | --- | --- |
| Base | Small operators | Single deployment, auth, workspaces, channels, scheduled posts, auto replies, filters, reports, webhook setup | 5-7 business days | Custom quote |
| Growth | Active teams | Base + production hardening, backup checklist, operator training, launch support | 7-10 business days | Custom quote |
| Enterprise | High-touch customers | Growth + custom workflow tuning, stricter rollout checklist, extended handover, support SLA negotiation | 10-20 business days | Custom quote |

## Included by Default

- Backend deployment on customer-owned infrastructure
- PostgreSQL and Redis wiring
- Environment setup and migration run
- API, worker, and scheduler process setup
- Healthcheck and smoke verification
- Delivery docs and handover checklist

## Optional Add-Ons

- Extra environment setup such as staging
- Extended maintenance retainer
- Additional operator training session
- Custom reporting or workflow adjustments inside MVP scope

## Out of Scope

- Public multi-tenant SaaS launch
- Frontend dashboard development
- Payment and subscription systems
- AI features
- CRM or campaign management
- Non-Rubika messaging platform expansion unless separately scoped

## Scope Guard

Any request outside the locked MVP modules must be quoted as a separate change request before implementation starts.
